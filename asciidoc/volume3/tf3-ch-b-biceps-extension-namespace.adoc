== BICEPS Extension Namespace

Extensions to the BICEPS Participant and Message Model defined in this specification utilize the XML Schema target namespace URI `urn:oid:1.3.6.1.4.1.19376.1.6.2.10.1.1.1`, of which the OID is further detailed in <<vol3_table_sdpi_biceps_extension_namespace>> and <<vol3_appendix_c_oids>>.

[#vol3_table_sdpi_biceps_extension_namespace]
.SDPi BICEPS Extension Namespace OID Assignment
[cols="2,3,1",options="autowidth, header"]
|===
| Primary identifier
| Concept description
| Secondary identifier

| `{SDPi-parent-oid}`
| Profile specific OID for SDPi
| sdpi

| `{SDPi-parent-oid}.*1*`
| Describes namespaces for different purposes as specified by its sub-nodes
| namespaces

| `{SDPi-parent-oid}.*1.1*`
| Extensions to the BICEPS Participant and Message Model
| biceps-extensions

| `{SDPi-parent-oid}.*1.1.1*`
| Major version 1 for extensions to the BICEPS Participant and Message Model.
  In order to avoid proliferation of OIDs below `{SDPi-parent-oid}.1.1`, versions are incremented only if incompatible changes are made to an extension.
| version1
